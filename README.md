# AffectiveSense Frontend
## Modern React UI — Real‑Time Emotion Classification Interface

A clean, futuristic, and minimal React interface designed to interact with the AffectiveSense Emotion Classification API. This frontend delivers a polished, enterprise‑grade user experience with webcam capture, live emotion updates, and a glass‑styled medical‑tech aesthetic that aligns with the AffectiveSense brand.

---

## ✨ Features
- Real‑time webcam preview with frame capture  
- Clean, modern, glass‑style UI  
- Responsive layout for desktop and mobile  
- API integration with the AffectiveSense backend  
- Smooth emotion result display with confidence values  
- Modular component structure for easy expansion  
- Production‑ready build optimized for cloud deployment  

---

## 🧠 Tech Stack

### Frontend
- React (Vite or CRA)  
- Modern CSS with glass‑morphism styling  
- Webcam capture component  
- Axios or Fetch for API communication  

### Backend Integration
- Connects to the AffectiveSense FastAPI backend  
- Sends frames or text for emotion prediction  
- Displays real‑time results in the UI  

### Deployment
- Azure Static Web Apps  
- Netlify / Vercel (optional alternatives)  

---

## 🚀 How It Works
- The user opens the React app.  
- The webcam component streams live video.  
- A frame is captured and sent to the backend API.  
- The backend preprocesses the image and runs the ML model.  
- The model returns emotion predictions with confidence scores.  
- The UI displays the detected emotion in a clean, modern results panel.
